# YTKJAutoHealthReport

烟台科技学院 数字烟科 健康上报模块自动化上报工具

## 简述

该脚本可以自动化完成繁琐的日常健康上报流程，解放双手和时间。  
由于烟科已不再使用该系统，现已开源。

## 依赖

* chrome-driver
* unirest-java
* yamlconfiguration
* jsoup
* gson
* lombok
* wxpusher-java-sdk
* baidu aip sdk
* apache commons-lang3
* Java-Markdown-Generator
* selenium-java
* commons-net
* easyocr (未使用)
